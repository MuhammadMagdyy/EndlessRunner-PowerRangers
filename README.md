# Endless Runner - Power Rangers

A fast-paced, high-action endless runner game inspired by the **Power Rangers** universe. Dodge obstacles, collect power-ups, and survive as long as possible.

![Gameplay Preview](Screenshots/gameplay_preview.png)

## 🚀 Features
- **Dynamic Movement:** Smooth lane-switching and jumping mechanics.
- **Thematic Assets:** Models and environments inspired by Power Rangers.
- **Performance Optimized:** Efficient object pooling for obstacles and path segments.
- **Cross-Platform:** Designed for both Mobile and Desktop play.

## 🛠️ Technical Setup
1. **Unity Version:** 2022.3+
2. **Installation:**
   ```bash
   git clone https://github.com/MuhammadMagdyy/EndlessRunner-PowerRangers.git
   ```
3. **Open:** Open the project via Unity Hub. (Note: The `Library` folder is ignored and will generate locally).
